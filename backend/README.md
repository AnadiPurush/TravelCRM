# TravelCRM

TravelCRM is a full-stack Customer Relationship Management (CRM) system designed for travel and holiday businesses. It helps manage leads, customers, agents, bookings, queries, and communication in one central platform.

## Project Structure
✔ JWT-based authentication  
✔ Stateless security layer  
✔ Role-based access control  
✔ Token expiry validation  
✔ Default super-admin user
